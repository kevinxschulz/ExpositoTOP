Detailed Description

Utility class for generating all subsets (power set) of a given set.

This generic class implements both Iterator and Iterable interfaces to enumerate all possible subsets of a provided set. It uses a BitSet to efficiently track subset membership and supports iteration over the power set. Useful for combinatorial algorithms and exhaustive search tasks.
