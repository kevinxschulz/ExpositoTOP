Detailed Description

Implements the Bellman-Ford algorithm for shortest path calculation.

This class provides functionality to compute shortest paths in a weighted directed graph using the Bellman-Ford algorithm. It manages the distance matrix, node and edge lists, and the resulting path and distance values. The class supports initialization with a distance matrix and node count, and exposes methods to solve the shortest path problem, retrieve computed distances, and access the optimal path and its value.
