Detailed Description

Utility class for reading and parsing TOPTW problem instances from files.

This class provides static methods to load Team Orienteering Problem with Time Windows (TOPTW) instances from a specified file path. It parses the input file, initializes the problem data, and computes the distance matrix required for further processing.

TOPTW top.TOPTWReader.readProblem ( String filePath)

Reads a TOPTW problem instance from a file.

Parameters
filePathPath to the input file containing the problem definition.
Returns
A TOPTW object initialized with the parsed data.

This method opens the specified file, reads the problem parameters and points of interest (POIs), sets their coordinates, service times, scores, and time windows, and calculates the distance matrix. If an error occurs during reading, the method prints the error and terminates the program.
