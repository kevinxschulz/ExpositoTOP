Detailed Description

Implements the GRASP metaheuristic for solving TOPTW instances.

This class provides methods for constructing and improving solutions to the Team Orienteering Problem with Time Windows (TOPTW) using the Greedy Randomized Adaptive Search Procedure (GRASP). It includes routines for greedy randomized construction, candidate selection (including fuzzy logic and alpha-cut strategies), and solution evaluation. The class supports iterative optimization, restricted candidate list management, and integration with solution and problem data.
